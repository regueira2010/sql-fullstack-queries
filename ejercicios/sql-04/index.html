Gestión de contenido multimedia mediante inserción masiva y análisis estadístico de temporadas y años de estreno.

Insertar datos para poblar el catálogo con diversidad de géneros y años.

2. Series con más de 3 temporadas

SELECT * FROM serie_netflix 
WHERE temporadas > 3 
ORDER BY anio_estreno DESC;

Filtrado de contenido extenso ordenado cronológicamente de forma descendente.

3. Identificación de la Serie más Antigua

SELECT MIN(anio_estreno) AS anio_antiguo 
FROM serie_netflix;

Uso de la función de agregación MIN para localizar el origen del catálogo.

4. Identificación de la Serie más Nueva

SELECT MAX(anio_estreno) AS anio_new 
FROM serie_netflix;

Uso de MAX para determinar el estreno más reciente en la base de datos.

5. Promedio de Año de Estreno (Redondeado)

SELECT ROUND(AVG(anio_estreno)) AS anio_promedio 
FROM serie_netflix;

Cálculo del promedio central de estrenos aplicando redondeo para obtener un año entero.

6. Promedio de Temporadas

SELECT ROUND(AVG(temporadas)) 
AS temporada_promedio FROM serie_netflix;

Cálculo de la media de duración de las series en el catálogo.

7. Series con 1, 2, 4, 5 o 7 temporadas

SELECT * FROM serie_netflix
WHERE temporadas IN (1, 2, 4, 5, 7)
ORDER BY temporadas ASC;

Filtrado por conjunto de valores específicos usando el operador IN.

8. Series que NO tienen 1, 2, 4, 5 o 7 temporadas

SELECT * FROM serie_netflix
WHERE temporadas NOT IN (1, 2, 4, 5, 7)
ORDER BY temporadas ASC;

Exclusión de grupos de datos específicos mediante NOT IN.

9. Borrado de Series (Post-2010)

DELETE FROM serie_netflix WHERE anio_estreno > 2010;

Ejecución de sentencia DELETE con filtro condicional de año.

Restauración de registros tras limpieza de tabla.

11. Inserción de Serie Específica (Doctor House)

INSERT INTO serie_netflix (nombre, temporadas, genero, anio_estreno) 
VALUES ('Doctor House', 8, 'Drama Médico', 2004);

Adición de un registro individual al catálogo existente.

12. Series Estrenadas entre 2005 y 2020

SELECT * FROM serie_netflix
WHERE anio_estreno BETWEEN 2005 AND 2020
ORDER BY anio_estreno ASC;

Uso del operador de rango BETWEEN para segmentación temporal.

13. Filtros por Texto

SELECT * FROM serie_netflix
WHERE nombre LIKE 'b%' OR nombre LIKE '%e'

Búsqueda de patrones mediante LIKE para nombres que inician con 'b' o terminan con 'e'.

14. Proyección: Año de Estreno + Temporadas

SELECT nombre, (anio_estreno + temporadas) AS calculo 
FROM serie_netflix WHERE (anio_estreno + temporadas) > 2010;

Operación aritmética entre columnas para generar una métrica combinada superior a 2010.
